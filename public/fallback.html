Troubleshooting Steps

  1. Check Earth Engine Authentication:

    Run the test script to verify Earth Engine connection:

    node test-connection.js
  2. Verify Google Maps API Key:

    Make sure your Google Maps API key is valid and has the necessary APIs enabled:

    • Maps JavaScript API
    • Places API
  3. Check Browser Console:

    Open your browser's developer tools (F12 or Ctrl+Shift+I) and check the console for specific error messages.

  4. Restart the Server:

    Try stopping and restarting the server:

    npm start

    or

    ./start.sh

Still Having Issues?

If you're still experiencing problems, try the following:

  • Clear your browser cache and cookies
  • Try a different browser
  • Check if your firewall is blocking any connections
  • Ensure you have the latest version of the code
